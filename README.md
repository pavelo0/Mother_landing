# Mother Landing

## Commit Message Prefixes

-   **layout** - Изменения в структуре страницы, блоках, сетке
-   **style** - Правки CSS/SCSS, шрифтов, цветов, отступов
-   **ui** - Изменения в интерфейсе (кнопки, формы, элементы)
-   **fix** - Исправление багов в верстке
-   **docs** - Обновление документации по стилям или компонентам
-   **chore** - Служебные задачи (сборка, конфиги, линтеры)
